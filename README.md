# Raytrophi

### 🇹🇷 Türkçe Açıklama

**Raytrophi**, fiziksel tabanlı bir ışın izleme (raytracing) motorudur. Gerçekçi materyal etkileşimleri, ışık saçılımı, gölgeler ve hacimsel efektleri hedefleyen modern bir render altyapısıdır. 

### Özellikler:
- Yüksek kaliteli PBR materyaller (Principled BSDF uyumlu)
- Gelişmiş ışık sistemleri: noktasal, yönsel, alan ışıkları
- CUDA destekli OptiX entegrasyonu
- Embree hızlandırmalı BVH yapısı (isteğe bağlı)
- Motion blur ve animasyon desteği
- Volumetrik efektlere açık yapı
- Blender ile entegre sahne dışa aktarımı (GLTF)

---

### 🇬🇧 English Description

**Raytrophi** is a physically-based raytracing engine designed for photorealistic rendering. It supports realistic material interaction, lighting, shadows, and volumetric effects.

### Features:
- High-quality PBR materials (Principled BSDF compatible)
- Advanced lighting: point, directional, area lights
- CUDA-based OptiX integration
- Optional Embree-accelerated BVH structure
- Motion blur and animation-ready
- Volumetric rendering structure
- GLTF export support from Blender
