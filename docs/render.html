3. Hardware Acceleration

OptiX (GPU) vs Embree (CPU)

OptiX: Uses NVIDIA RT Cores for extremely fast ray traversal. Supports full scene geometry, motion blur, and instancing.
Embree: Uses CPU AVX/SSE instructions. Used as a fallback or for non-NVIDIA GPUs.

Switching modes requires a full scene rebuild. We recommend sticking to OptiX on RTX cards.
