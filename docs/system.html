1. Project Management

RayTrophi uses a proprietary .rtp (RayTrophi Project) format.

File Menu

  • New Project (Ctrl+N)
  • Open Project (Ctrl+O)
  • Save Project (Ctrl+S)
  • Import Model (Ctrl+I)
