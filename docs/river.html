Please save a showcase render or screenshot of a River as manual/images/river_header.jpg

The River Tool allows you to draw procedural rivers directly onto the terrain using Cubic Bezier Splines.

1. Spline Editing

Please save a screenshot of the River Spline / Gizmo as manual/images/river_spline.png
Action Description
Add Point Adds a new control point at the end of the river based on the current camera view.
Move Point Select a point in the list or 3D view and use the Transform Gizmo to position it.
Subdivide Adds a new point between two existing selected points for finer control.

2. Mesh & Flow Physics

Please save a screenshot of the River Properties Panel as manual/images/river_physics.png
Parameter Description
Width Base width of the river.
Resolution Number of polygons along the length.
Flow Speed How fast the water texture scrolls along the UVs.
Turbulence Distortion amount for the flow vectors.
