GPU-accelerated heightmap engine with hydraulic erosion and vegetation painting.

1. Hydraulic Erosion

Erosion UI

Simulates water droplets dissolving and transporting soil.

Parameter Description
Rain Amount Droplets per cycle.
Solubility How easily soil dissolves.
Sediment Capacity Max soil carried by water.
Evaporation Drying rate.
Gravity Speed of droplet descent (affects pit formation).

2. Foliage System (New)

Foliage / Brush UI

Paint vegetation directly onto the terrain mesh.

Brush Settings

Action Description
Left Click Paint instances within radius.
Shift + Click Erase instances within radius (or use Erase Mode).

Parameters

Parameter Description
Radius Size of the brush cursor.
Density Number of instances spawned per brush stroke.
Align to Normal If checked, trees point perpendicular to the slope.
Random Rot Adds random Y-rotation for variety.
