Advanced physically-based hair system featuring Marschner scattering and interactive grooming tools.

1. Interactive Grooming & Brushes

Grooming Brush UI

Shape and style hair strands directly in the viewport using intuitive brush tools. Brushes support custom radius, strength, and falloff settings.

Tool Action
Comb Guides strands in the direction of the brush stroke.
Length / Cut Add length or trim hair strands within the brush radius.
Density Interactively add or remove strands based on surface area.
Clump / Braid Create natural clustering or advanced rotational braiding patterns.
Smooth Relaxes strand orientations for more uniform distributions.

2. Hair Materials (Marschner BSDF)

Hair Material UI

Physically based shading via the Marschner model, providing realistic light transport through hair fibers.

Parameter Description
Melanin Natural pigment concentration (0.0 to 1.0).
Roughness Surface micro-detail for specular highlights (R lobe).
Absorption Controls light energy loss as it passes through the fiber.
Specular Shift Highlights offset due to the scale-like structure of hair.

3. Layer Management & Selection

Hair Layer / UI List

RayTrophi allows for a non-destructive multi-layer grooming workflow:

  • Layer List: Create, delete, and hide individual hair grooms.
  • Renaming: Double-click any layer name to rename it instantly.
  • Shared Materials: Assign shared material presets across different grooms for consistency.
  • Mesh Sync: Grooms are automatically linked to their parent mesh transforms.
