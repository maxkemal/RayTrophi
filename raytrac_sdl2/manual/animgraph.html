Anim Graph & State Machines

The Animation Graph allows for complex character logic using State Machines and Blend Trees.

1. State Machine

Organize animations into logical states (e.g., Idle, Walk, Jump).

Please save a screenshot of the Anim Graph State Machine as manual/images/anim_fsm.png

Transitions: Connect states with arrows. Click an arrow to define transition conditions.

  • Has Exit Time: Transition happens automatically when animation finishes.
  • Conditions: Transition happens when a parameter meets a criteria (e.g., Speed > 0.1).

2. Parameters

Variables that control the graph logic. Listed in the left panel of the Anim Graph.

  • Float: Continuous values (Speed, Direction).
  • Int: Discrete states (WeaponID).
  • Bool: Binary states (IsGrounded, IsDead).
  • Trigger: One-shot events (Attack, Jump).
