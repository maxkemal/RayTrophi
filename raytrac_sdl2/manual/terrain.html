Terrain & Erosion

⛰️
Please save a showcase render of the Terrain as manual/images/terrain_header.jpg

RayTrophi features a powerful heightmap-based terrain engine with GPU-accelerated sculpting and hydraulic erosion simulation.

1. Hydraulic Erosion Simulator

The erosion system simulates thousands of water droplets falling onto the terrain, dissolving soil, transporting it downhill, and depositing it in pools.

Please save a screenshot of the Erosion Panel as manual/images/terrain_erosion.png
Parameter Description
Rain Amount Number of water droplets to simulate per cycle.
Solubility How easily the terrain soil dissolves into water.
Sediment Capacity Maximum amount of soil a droplet can carry.

2. Node Graph & Generation

Combine multiple heightmaps and procedural noises using the Node Graph editor.

Please save a screenshot of the Terrain Node Graph as manual/images/terrain_nodegraph.png

Common Nodes: Perlin Noise, Worley Noise, Mix, Curve, Erosion.
